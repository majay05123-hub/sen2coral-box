SWAM Operator - Installation on Linux - Anaconda


  1. Install Anaconda 3 v4.3.1:

    See

  2. Configure Anaconda to use python 3.4: conda install python=3.4.0
  3. Install matplotlib: conda install matplotlib
  4. Install xmltodict from the command prompt : conda install xmltodict
  5. Install rasterio: conda install rasterio
  6. Install future : pip install future
  7. pip install spectral
  8. Install Sambuca & Sambuca_core:
    1. Create a directory called "sambuca_project"
    2. Download Sambuca and Sambuca_core (link?) and copy them in "Sambuca_project"
    3. Open an Anaconda command prompt
    4. Using the Command Prompt, go into the directory Sambuca_project/Sambuca_core and type "python setup.py develop"
    5. Using the Command Prompt, go into the directory Sambuca_project/Sambuca and type "python setup.py develop".
  9. Configure python in SNAP via command line:

    snappy-conf [PathToAnacondaInstallationFolder]/python.exe

  10. Set enviroment variables on the .bashrc file:
    1. export PYTHONPATH=[sambuca_folder]: [snap-python folder]: [sambuca_core folder]:[anaconda3 folder]/lib/python3.4:[anaconda3 folder]/lib/python3.4/site-packages:[anaconda3 folder]/lib:$PYTHONPATH
    2. export PATH=[anaconda3 folder]/bin:$PATH
    3. export PYTHONHOME=[anaconda3 folder]/lib/python3.4/
